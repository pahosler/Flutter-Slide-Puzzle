# Flutter Slide Puzzle

## Getting Started

Clone this repo

`$ cd flutter_slide_puzzle`

`$ flutter packages get`

`$ flutter run`

## About
A simple slide puzzle game.

Slide the tiles to arrange the numbers into proper order.
